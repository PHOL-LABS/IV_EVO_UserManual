\chapter{Package contents}\label{ch:package}

Each IV-Indicators Nano shipment contains the essentials required for installation:
\begin{itemize}
    \item Button gauge or gauge set (depending on the ordered variant).
    \item Socket for connection to the vehicle harness.
    \item External programming tool with USB connectivity.
\end{itemize}
